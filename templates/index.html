API Dokumentacija

Dostupni endpointi:

  • GET /api/videos - Dohvati sve spotove
  • GET /api/videos/{id} - Dohvati spot po ID-u
  • POST /api/videos - Dodaj novi spot
  • PUT /api/videos/{id} - Ažuriraj postojeći spot
  • DELETE /api/videos/{id} - Obriši spot
  • GET /api/videos/by-artist/{artist} - Filtriraj po izvođaču
  • GET /api/videos/by-year/{year} - Filtriraj po godini
  • GET /api/videos/most-viewed - Dohvati najgledanije spotove
  • GET /api/docs - Dohvati OpenAPI specifikaciju
